# 💳 SwiftQueue Banking App

A full-stack backend banking system built with **Java Spring Boot**, designed to handle user authentication, banking operations, appointments, queue management, and branch services.

---

## 🚀 Features

### 🔐 Authentication & Security
- User registration and login
- JWT-based authentication
- Role-based access control (Customer, Staff, Admin)
- Secure password hashing

---

### 🏦 Banking Core
- User account management
- Transaction handling (deposit, transfer, etc.)
- Wallet/account operations

---

### 📅 Appointment System
- Book bank appointments
- Manage appointment status
- Staff scheduling support

---

### 🧾 Queue Management System
- Join virtual queues
- Priority-based queue handling
- Ticket generation and tracking
- Real-time queue status updates

---

### 🏢 Branch & Services
- Branch management
- Service listings per branch
- Staff assignment to services

---

## 🛠️ Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- JPA / Hibernate
- PostgreSQL / MySQL
- Maven
- Docker

## 🏗️ Architecture

This project follows a **layered architectural design** based on Spring Boot best practices:

- Controller Layer – Handles HTTP requests and responses  
- Service Layer – Contains business logic  
- Repository Layer – Manages database operations  
- Model Layer – Defines entity structures  
- DTO Layer – Handles data transfer between layers  
- Config Layer – Security and application configuration  
- Exception Layer – Global exception handling
